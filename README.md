# 🌾 Crop Price Prediction Web App

A **Machine Learning + Django web application** that predicts crop prices based on:

- Commodity (Crop)
- State
- Market Name
- Year
- Month  

Users can enter **real-world inputs like "Wheat"** instead of encoded numbers.

---

## 🚀 Features

- ✅ Predict crop prices using ML model  
- ✅ Supports **categorical input (Wheat, Rice, etc.)**  
- ✅ Dynamic dropdowns (auto-filled from dataset)  
- ✅ Clean Django-based web interface  
- ✅ Error handling for invalid input  
- ✅ Model + encoders bundled together  

---

## 🧠 Tech Stack

- Backend: Django  
- Machine Learning: Scikit-learn  
- Language: Python  
- Frontend: HTML, CSS  

---

## 📊 Model Details

- Algorithm: Random Forest Regressor  
- Features used:
  - Commodity
  - STATE
  - Market Name
  - Year
  - Month  

- Target:
  - Modal Price
